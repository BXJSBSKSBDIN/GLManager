# GLManager

## Modo de instalacao manual

* Instalando GIT 

```
sudo apt-get update && sudo apt-get install git -y
```

* Instalando script
```
cd ~/
git clone https://github.com/DuTra01/GLManager.git
cd GLManager
pip3 install -r requirements.txt
```

* Execute o script
```
cd ~/GLManager
python3 -m app
```
